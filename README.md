# 🧠 Online Examination System

A desktop-based **Online Examination System** built using **Python, Tkinter, and SQLite**.  
This application enables admins to create exams and students to attend them with a timer and instant evaluation.

---

## 📌 Features

### 👨‍💼 Admin
- Create student accounts
- Create exams with duration
- Add, edit, and delete questions
- View all student results
- Analyze scores
- Edit admin profile

### 👨‍🎓 Student
- Login securely
- Attempt exams with timer
- Navigate between questions
- Auto-save answers
- Submit exam anytime
- View detailed results

---

## 🛠️ Technologies Used

- Python
- Tkinter (GUI)
- SQLite3 (Database)
